# starter-qwik

An opinionated starter for creating a Qwik library.

## Features

- Using [Vite+](https://viteplus.dev/) for bundling, formatting, linting, and testing.
- Only supports ESM, no CJS.
- Automatic testing and publishing with GitHub Actions.

## Note for developers

I highly recommend using [Vite+](https://viteplus.dev/) for managing Node.js runtime.

The Oxc formatter config is **a personal config** so it may not be suitable for everyone. You can modify it according to your preferences.
